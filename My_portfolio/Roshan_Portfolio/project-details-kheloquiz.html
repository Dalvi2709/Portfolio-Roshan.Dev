KheloQuiz – AI Powered Quiz Generation Platform

An innovative AI-driven platform that instantly generates personalized quizzes based on user-selected topics and subtopics. Built with Django and Google Gemini API, it delivers dynamic, engaging quizzes through a user-friendly interface and real-time feedback.

Project Overview

KheloQuiz bridges the gap between static quiz systems and personalized learning needs by using AI to dynamically create quizzes tailored to individual topics. Users can generate quizzes instantly, track performance, and enhance learning through real-time feedback and gamified elements.

Objectives

  • Automate the quiz creation process using AI.
  • Provide personalized quizzes for specific topics and subtopics.
  • Ensure an interactive and user-friendly learning experience.
  • Integrate scalable and secure backend architecture for seamless functionality.

Key Features

  • AI-powered question generation using Google Gemini API.
  • Secure authentication and user profile management.
  • Gamified quiz experience with real-time scoring and progress tracking.
  • Performance reports for post-quiz analysis.
  • Responsive design for desktop and mobile users.

Technology Stack

  • Frontend: HTML, Tailwind CSS
  • Backend: Django (Python)
  • Database: PostgreSQL
  • AI Integration: Google Gemini API
  • Tools: Visual Studio Code, Postman, Figma, Docker

Methodology

The backend is developed using Django REST Framework, handling API requests and integrating AI functionalities. Postman was used for API testing, while CI/CD pipelines ensured automated deployment. The frontend leverages Tailwind CSS for responsive UI, and quizzes are generated in real-time using Gemini API.

Applications

  • Education – Custom quizzes for e-learning platforms and schools.
  • Corporate Training – Tailored skill assessments and performance tracking.
  • Edutainment – Fun learning experiences for children and casual learners.
  • Self-Assessment – Track personal progress in chosen topics.

Conclusion

KheloQuiz offers a dynamic and engaging way to learn, test, and track knowledge. With AI integration, a responsive interface, and secure backend architecture, it stands as a future-ready solution for education and skill enhancement.
